How exactly does the naming convention work?

.pg-<VIEWPORT>-<NUMERATOR>-<DENOMINATOR>

There is no maximum amount of columns.

  • xs - Extra Small, for mobile (> 0)
  • sm - Small, for tablet (≥ 768px)
  • md - Medium, for desktop (≥ 992px)
  • lg - Large, for wide monitors (≥ 1200px)

If you wanted to make a 4 column grid for regular desktop:

grid medium 1/4
.pg-md-1-4

2 column grid for mobile:

grid extra small 1/2
.pg-xs-1-2
